IT Engineer Portfolio

백엔드 개발과 시스템 운영 관점에서 서비스를 바라보는 IT엔지니어 지망생입니다.

기능 구현에 그치지 않고, 서비스가 실제 환경에서 어떻게 동작하고 유지되는지를 함께 고민하며 개발하고자 합니다.

기술 스택

  • Language: Java, JavaScript, Python
  • Backend: Spring Boot, REST API
  • Database: MySQL, Oracle
  • Infra / DevOps: Linux, Docker
  • Monitoring: Prometheus, Grafana, cAdvisor
  • Tools: Git/GitHub, Notion, Teams, Slack

프로젝트

Docker 기반 컨테이너 모니터링 시스템

Linux 환경에서 Docker 컨테이너로 실행되는 서비스의 자원 사용 상태를 모니터링하고, 실행 환경별 성능 특성을 비교한 프로젝트입니다.

Prometheus, Grafana, cAdvisor를 활용해 CPU·메모리·네트워크 지표를 수집하고 시각화했습니다.
